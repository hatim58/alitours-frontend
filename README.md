# Ali Tours & Travels Website

A professional travel agency website for Ali Tours & Travels, specializing in any tour packages.

## Features

- Responsive design that looks great on all devices
- Tour package listings with detailed information
- Day-by-day itinerary display
- Booking form system
- Contact page with WhatsApp integration
- User registration and login
- Admin panel for managing bookings and packages

## Technologies Used

- React 18
- TypeScript
- TailwindCSS
- React Router Dom
- React Hook Form
- Lucide React for icons

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`

## Hosting Instructions

To deploy this website to a hosting service:

1. Build the project: `npm run build`
2. Upload the contents of the `dist` folder to your web hosting server
3. If using Netlify, Vercel, or similar services, you can connect your Git repository for automatic deployments
